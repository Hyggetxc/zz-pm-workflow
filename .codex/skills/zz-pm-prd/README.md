# zz-pm-prd

一个可复用的 PRD 专项技能，适用于从首次评审稿到正式交付版 PRD 的分阶段输出。

## 目录结构

- `SKILL.md`：技能主说明
- `references/`：输入契约、风格模板、初评版与正式版 PRD 模板
- `examples/`：反例与结构示例

## 适用场景

- 已有原型、截图、规则或历史文档，需要快速生成结构化 PRD
- 需要先出一版可评审文档，再升级为正式交付版 PRD
- 希望统一“功能清单、页面详设、待确认项、自检”的写法

## 安装

如果你使用支持 GitHub 路径安装技能的工作流，可按仓库路径安装本技能。

示例思路：

- 仓库名与技能名保持一致
- 仓库根目录直接放技能文件，避免额外嵌套层级

## 说明

- 发布版已经去除了项目专名和环境绑定描述
- 默认优先本地 `docx` 输出，必要时可降级为 `Markdown`
- 若你需要绑定特定在线文档平台，请在自己的项目中再做二次扩展

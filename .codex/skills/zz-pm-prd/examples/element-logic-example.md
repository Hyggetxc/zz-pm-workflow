# 元素与逻辑示例

## 列表页写法示例

### 正例

| 功能模块 | 元素 | 实现归属 | 逻辑说明 |
| ---- | ---- | ---- | ---- |
| 课程列表 | 展示范围 | 前后端 | 展示当前筛选条件下的全部课程数据，一行对应一门课程。 |
| 课程列表 | 展示顺序 | 后端 | 按课程创建时间从近到远展示。 |
| 课程列表 | 课程名称，文本 | 前端 | 名称过长时省略展示，鼠标悬停展示全部。 |
| 课程列表 | 主讲老师，文本 | 前后端 | 展示老师姓名；若支持按 ID 搜索，则在筛选区说明，不在列表字段中重复解释。 |
| 课程列表 | 排课进度，进度条 + 文本 | 前后端 | 展示已排课课时数 / 总课时数；总课时为 0 时展示“暂无排课”。 |
| 课程列表 | 课程状态，状态标签 | 前后端 | 取值：草稿、已上架、已下架。 |
| 课程列表 | 课程名称点击 | 前端 | 点击后进入课程详情。 |
| 课程列表 | 操作区 | 前后端 | 展示编辑、上架、下架、删除；具体可操作范围由课程状态决定。 |
| 分页 | 分页器 | 前后端 | 仅当列表总数大于单页条数时展示；切换筛选条件后回到第一页。 |

## 字段规则示例

1. 错误集中。
   a. 统计口径：仅统计已作答人数，未作答不进入错误分布。
   b. 判定规则：错误人数为 0 时不展示“错误集中”。
   c. 集中规则：错误选项中人数最多的选项人数 >= 2 且占错误人数比例 >= 50% 时，展示“错误集中在 X”。
   d. 并列规则：若两个错误选项并列第一，则展示“错误集中在 A/C”。

2. 直播时间。
   a. 展示格式：日期标签 + 开始时间 - 结束时间。
   b. 日期标签规则：当天展示“今天”，次日展示“明天”，同年其他日期展示“M月D日”，跨年展示“YYYY年M月D日”。
   c. 时间格式规则：24 小时制 HH:mm，不展示秒。
   d. 跨天规则：结束时间跨天时，补充次日日期标签。
